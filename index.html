Google Calendar API Key
Go to console.cloud.google.com → APIs & Services → Credentials → Create API Key. Enable the Google Calendar API in the library. Restrict the key to your GitHub Pages domain under "Website restrictions".
Calendar IDs (one per line)
In Google Calendar, go to Settings → your calendar → Integrate calendar and copy the Calendar ID. One per line. Each calendar gets its own color automatically.
Example: you@gmail.com or family@group.calendar.google.com
